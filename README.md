# Restaurant QR Order

Website đặt món ăn bằng mã QR sử dụng PHP và MySQL.

## Chức năng chính

- Quét mã QR để xác định bàn
- Hiển thị danh sách món ăn
- Chọn món và nhập số lượng
- Quản lý giỏ hàng
- Xác nhận đặt món
- Lưu đơn hàng vào MySQL

## Công nghệ sử dụng

- PHP
- MySQL
- HTML
- CSS
- JavaScript
- XAMPP
- Git và GitHub
